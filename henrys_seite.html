Nesting

The SCSS code will follow the same visiual hierarchy like the html. If a code for one element is written inside another element it will behave like :
Outer Element (Parent) > Inner Elemtent (Child)

Example:

CSS:

.content {
     font-size: 25px;
     color: green;
}

.content p {
     color: red;
}
>
SCSS:

.content {
     font-size: 25px;
     color: green;

            p {
                  color: red;
               }
}
