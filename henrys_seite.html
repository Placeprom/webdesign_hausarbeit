Nesting

The SCSS code will follow the same visiual hierarchy like the html. If a code for one element is written inside another element it will behave like :
Outer Element (Parent) > Inner Elemtent (Child)

Example:

//CSS:

.content {
     font-size: 25px;
     color: green;
}

.content p {
     color: red;
}
>
//SCSS:

.content {
     font-size: 25px;
     color: green;

            p {
                  color: red;
               }
}

Nested Properties

You can also nest proberties which share the same "forename". Therefore you don't need to write repeadly the identical proberty over and over again.

Example:

//CSS:

.class {
     font-family: Helvetica;
     font-size: 18px;
     font-weight: bold;
}
>
//SCSS:

.class {
     font- {
          family: Helvetica;
          size: 18px;
          weight: bold;
      }
}
