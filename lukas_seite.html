Variables and Operators

Variables

What are Variables

With Variables you are able to store information that you can reuse. This can make your CSS code shorter and much easier to maintain, since you just have to change the value at a singel position in your code.
Altough it's called variable, it's much more used as a constant, since you usually won't change the value of a self declared variable.

How to use Variables

Now we will show you the syntax used in SCSS. Once you have understood the snytax, you will be able to declare your own variables. To declare a variable, you just have to place a $ sign followed by the name you want your variable to have. Then you add : followed by the string (the information) you want to save. An example is shown in Example for Variables.

Example for Variables


$myWidth: 100px;
$myBorder: 1px solid green;

myBlock {
    width: $myWidth;
    border: $myBorder;
}

In this Example we saved the settings for a border in $myBorder and the settings for our width in $myWidth.

Operators

What are Operators

Operators can be used for linguistic, logical or mathmatical operations. For example multiplying two Number with each other, concatinating two strings together or comparing two numbers with each other.
With SCSS you can finally use lots of operators without using calc() or the use of any other functions. In the following talbe you can observe operators you can use in SCSS.

Operators in SCSS
Operator Function
A + B Adds A and B together (if numbers) or concatinates B to A (if strings)
A - B Subtracts the B from A
A / B Devides A by B
A * B Multiplies A and B
A and B Returns true if A and B are true (else false)
A or B Returns true if at least one of A and B is true (else false)
not A Returns true if A is false and false if A is true (else false)
A == B Returns true if the A is equale to B (else false)
A != B Returns true if A is diffrent from B (else false)
A < B Returns true if A is smaller than B (else false)
A > B Returns true if B is smaller than A (else false)
A <= B Returns true if A is smaller than or equal to B (else false)
A >= B Returns true if B is smaller than or equal to A (else false)

How to use Operators

Operators are very easy to use. You simply have to add the operator you want to use inbetween the objects you want to use it on. An exception is the not operator, which only can be used in front of a singel object.
If you want to use an operator within a string, you can do that by putting both objects and the operator into curly brackets with a hashtag in front. Like this: #{A + B}.

Example for Operators


MyBox{
    width: calc(50% + 20px);
    height: calc(100px / 6 );
}

With SCSS 


MyBox{
    width: 50% + 20px;
    height: 100px / 6;
}
