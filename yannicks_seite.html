Inheritance & Extend

Inheritance is one of the most important features of Sass. With the @extend keyboard you are able to share CSS properties between different selectors. This means, that you can write a base-class for common styles which you then exted to specific needs of more specific classes.

Inheritance goes hand in hand with placeholder classes, another feature we will therefor be using right now. Placeholder classes only print if they are extended.

Example

LetÂ´s see how this works in an example:

%base-class {
border: 1px solid;
padding: 20px;
color: red;
}

%another-base-class{
border: 5px solid;
padding: 50px;
color: blue;
}

.extended-one{
@extend %base-class;
border-color: green;
}

.extended-two{
@extend %base-class;
border-color: blue;
}

Notice how in the above example only %base-class would be printed on screen. %another-base-class would not be printed since it is not extended.
