Inheritance & Extend

Inheritance is one of the most important features of Sass. With the @extend keyboard you are able to share CSS properties between different selectors. This means, that you can write a base-class for common styles which you then exted to specific needs of more specific classes.

Inheritance goes hand in hand with placeholder classes, another feature we will therefor be using right now. Placeholder classes only print if they are extended.

Example

LetÂ´s see how this works in an example:

%base-class {
border: 1px solid;
padding: 20px;
color: red;
}

%another-base-class{
border: 5px solid;
padding: 50px;
color: blue;
}

.extended-one{
@extend %base-class;
border-color: green;
}

.extended-two{
@extend %base-class;
border-color: blue;
}

Notice how in the above example only %base-class would be printed on screen. %another-base-class would not be printed since it is not extended.

Animation

Let us now have a look at an animation, which shows how inheritance works:

%base-class
.base-class-extended
  1. At first we have our %base-class which has the CSS property background-color: red; and the border-radius: 4px;
  2. We then extend our new class .extended-class with the keyword @extend %base-class;
  3. We add the property boder-style: solid; to the class .extened-class we also add background-color:yellow; and overwrite the red background.

Note: In fact the box with .base-class-extended would have a solid border and a yellow background when it is created (it does not transition to yellow).The animation is just a way to show it to you.
